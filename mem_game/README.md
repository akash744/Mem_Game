# mem_game

A Flutter project.

## Resources 

### Download the Font Files

Grab the font files from https://drive.google.com/file/d/1BVvL149gPGjIWf3pv961Fk6xb46to7FI/view?usp=sharing 

##### Code for pubspec.yaml
```yaml
  fonts:
    - family: Pacifico
      fonts:
        - asset: fonts/Pacifico-Regular.ttf
```
